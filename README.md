# mutual-selection-system

### 依赖：

* vue，框架
* vuex，状态管理器
* axios，网络请求库
* element-ui，UI框架
* nprogress，
* vue-puzzle-vcode，登录二维码插件
* vue-router，路由管理

### 安装：

首先安装依赖，会在目录下生成node_modules文件夹

```
npm install
```

启动本地服务器

```
npm run serve
```

看到如下结果，说明编译成功
